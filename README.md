# gf-micro

#### 介绍
goframe微服务框架实现，包括3个系统：web服务入口、用户中心微服务和博客中心微服务。
非功能性设计包括：单元测试、链路追踪、业务指标统计、日志规范、代码规范扫描、docker部署、蓝绿发布
